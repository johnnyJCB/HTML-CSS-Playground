Question 2:

2. Review the recommended web design practices from this chapter. Use the Web Design Best Practices Checklist (Table 5.1 ) to evaluate the JavaJam site that you created in earlier chapters. Cite three design practices that have been well implemented. Cite three design practices that could be implemented in a better way. How else would you improve the website?

In my JavaJam site, I have a consistent site/header logo across all the views, starting from the home page, to the menu, and music page. I also made good use of basic design principles: repetition, contrast, proximity, and alignment by primarily using a CSS file for formatting.
As far as navigation, the tab links are clearly and consistently labeled which makes it easy for the user to browse through. In regards to functionality, all the links work appropriately.
Among the things that could be improved are adding information as to when was the time the page was last updated, making sure that the views on the webpage work well with mobile devices, and I could also add hyperlinks to downloads for media plug-ins.
